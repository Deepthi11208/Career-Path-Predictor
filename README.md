# Career Path Predictor 🎯

An AI-powered web application that predicts suitable career paths based on your skills, interests, and educational background.

## 🚀 Features

- Predicts career paths based on user input
- Built with React for a smooth user experience
- Simple and intuitive UI

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS
- **Package Manager:** npm

## ⚙️ Installation & Setup

1. Clone the repository
```bash
   git clone https://github.com/Deepthi11208/Career-Path-Predictor.git
```

2. Navigate to the project folder
```bash
   cd Career-Path-Predictor
```

3. Install dependencies
```bash
   npm install
```

4. Run the app
```bash
   npm start
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📌 Usage

- Enter your skills and interests
- Click Predict
- View your recommended career paths
